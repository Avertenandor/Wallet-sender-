# WalletSender Modular - Контекстная сводка проекта

**Дата создания:** 6 сентября 2025  
**Версия проекта:** 2.1.1  
**Статус:** Production Ready

## 📋 Обзор проекта

**WalletSender Modular** - это модульное приложение для работы с блокчейном BSC (Binance Smart Chain), предназначенное для массовой рассылки токенов, автоматических покупок/продаж и управления кошельками.

## 🔧 Ключевые компоненты

### 1. JobRouter & JobEngine
- **Назначение:** Асинхронная обработка задач
- **Функции:** Очередь транзакций, колбеки, отмена задач
- **Статус:** ✅ Полностью интегрирован

### 2. NonceManager
- **Назначение:** Управление nonce для предотвращения конфликтов
- **Функции:** Автоматическое управление nonce, предотвращение дублирования
- **Статус:** ✅ Полностью интегрирован

### 3. BscScanService
- **Назначение:** Работа с Etherscan V2 API
- **Функции:** Ротация API ключей, rate limiting, получение данных о транзакциях
- **Статус:** ✅ Полностью интегрирован

### 4. RPC Pool
- **Назначение:** Управление множественными RPC подключениями
- **Функции:** Автоматическое переключение, балансировка нагрузки
- **Статус:** ✅ Полностью интегрирован

## ⚠️ Известные проблемы

### 1. Проблема с пулом WBNB
**Статус:** Документировано  
**Описание:** Отсутствует прямой пул ликвидности для WBNB к некоторым токенам  
**Влияние:** Может потребоваться использование промежуточных пулов (например, WBNB → USDT → PLEX ONE)  
**Решение:** Приложение автоматически определяет оптимальные пути обмена

### 2. Известные пулы PLEX ONE:

## 🚀 Функциональность

### Автоматические покупки (Auto Buy Tab)
- Поддержка покупки за BNB и USDT
- Автоматическое определение путей обмена
- Проверка ликвидности перед покупкой
- Настраиваемые интервалы и лимиты

### Автоматические продажи (Auto Sales Tab)
- Мониторинг балансов токенов
- Автоматическая продажа при достижении порогов
- Поддержка циклических продаж
- Настраиваемые slippage и минимальные цены

### Массовая рассылка (Mass Distribution Tab)
- Отправка токенов на множественные адреса
- Поддержка различных токенов
- Управление газом и nonce
- Отслеживание прогресса

### Система наград (Rewards Tab)
- Автоматическое начисление наград
- Настраиваемые условия и суммы
- Интеграция с JobRouter
