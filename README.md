# WalletSender Modular v2.3.0

**Продвинутый модульный инструмент для работы с токенами BSC и автоматизированной торговлей**

[![Version](https://img.shields.io/badge/version-2.3.0-blue.svg)](https://github.com/Avertenandor/Wallet-sender-)
[![Python](https://img.shields.io/badge/python-3.8+-green.svg)](https://python.org)
[![License](https://img.shields.io/badge/license-MIT-orange.svg)](LICENSE)

## 🚀 Основные возможности

### 🔄 Автоматизированная торговля
- **Автопокупка токенов**: Автоматическая покупка PLEX ONE за BNB/USDT
- **Автопродажа токенов**: Мониторинг и автоматическая продажа при достижении целей
- **Умное управление nonce**: Продвинутая система управления nonce для надежных транзакций

### 📊 Анализ и мониторинг
- **Анализ транзакций**: Детальный анализ транзакций BSC
- **Мониторинг балансов**: Отслеживание балансов токенов в реальном времени
- **История операций**: Полная история всех операций

### 🎁 Система наград
- **Автоматические награды**: Награждение активных пользователей
- **Массовая рассылка**: Отправка токенов множеству адресов
- **Настраиваемые конфигурации**: Гибкие настройки наград

### 🔧 Технические особенности
- **Модульная архитектура**: Чистая архитектура с разделением ответственности
- **Множественные RPC**: 10+ BSC RPC endpoints для максимальной надежности
- **Продвинутый NonceManager**: Ticket-based система управления nonce
- **Обработка ошибок**: Комплексная обработка всех типов ошибок

## 📋 Требования

## 🛠️ Установка

1. **Клонируйте репозиторий:**
```bash
git clone https://github.com/Avertenandor/Wallet-sender-.git
cd Wallet-sender-/WalletSender_Modular
```

2. **Установите зависимости:**
```bash
pip install -r requirements.txt
```

3. **Запустите приложение:**
```bash
python main.py
```

## 🔄 Автоматизированная торговля

### Автопокупка
- **Поддерживаемые токены**: PLEX ONE
- **Валюта покупки**: BNB, USDT
- **Маршруты обмена**:
  - BNB → USDT → PLEX ONE (через USDT)
  - USDT → PLEX ONE (прямой)
- **Защита от проскальзывания**: 1% slippage tolerance
- **Умное управление nonce**: Автоматическое разрешение конфликтов nonce

### Автопродажа
- **Мониторинг балансов**: Проверка каждые 30 секунд
- **Heartbeat механизм**: Логирование активности каждые 30 секунд
- **Изоляция ошибок**: Ошибки отдельных транзакций не останавливают мониторинг
- **Retry логика**: Автоматические повторы при ошибках

## 🛡️ Безопасность

- **Приватные ключи**: Безопасное хранение и использование
- **Seed phrases**: Поддержка BIP44 derivation path
- **Проверка контрактов**: Валидация существования контрактов
- **Checksum адреса**: Правильное форматирование адресов

## 📊 Мониторинг и логирование

- **Детальные логи**: Все операции логируются с временными метками
- **Статус мониторинг**: Реальное время статуса RPC, API, очереди
- **Диагностика**: Встроенные инструменты диагностики
- **История транзакций**: Полная история всех операций

## 🚨 Устранение неполадок

### Проблемы с nonce
- Приложение автоматически обрабатывает все типы nonce ошибок
- NonceManager обеспечивает правильную последовательность транзакций
- Fallback механизм при ошибках NonceManager

### Проблемы с подключением
- Множественные RPC endpoints для надежности
- Автоматическое переключение при недоступности
- Таймауты и retry логика

### Проблемы с транзакциями
- Детальное логирование всех операций
- Анализ причин неудачных транзакций
- Автоматические повторы при ошибках
