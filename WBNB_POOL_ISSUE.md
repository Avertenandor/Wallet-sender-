# Проблема с пулом WBNB - Документация

**Дата:** 6 сентября 2025  
**Статус:** Документировано  
**Приоритет:** Информационный

## 📋 Описание проблемы

В проекте WalletSender Modular обнаружена проблема с отсутствием прямого пула ликвидности для Wrapped Binance Coin (WBNB) к некоторым токенам, что может влиять на операции обмена.

## 🔍 Детали проблемы

### Что такое WBNB?
WBNB (Wrapped BNB) - это обернутая версия нативного токена BNB в сети BSC, используемая для взаимодействия с DeFi протоколами, такими как PancakeSwap.

**Адрес WBNB:** `0xbb4CdB9CBd36B01bD1cBaEBF2De08d9173bc095c`

### Проблема
При попытке прямого обмена некоторых токенов через WBNB может возникать ошибка "пул не найден" или "недостаточная ликвидность", что указывает на отсутствие прямого пула ликвидности.

## 🛠️ Реализованные решения

### 1. Автоматическое определение путей обмена

В файле `auto_buy_tab.py` реализована логика автоматического поиска оптимальных путей:

```python
def _check_known_pools(self, amount_in: int, buy_with: str = 'USDT') -> List[int]:
    """Проверяет известные пулы PLEX ONE"""
    if buy_with == 'USDT':
        # Проверяем прямой путь USDT -> PLEX ONE
        pool1_path = [CONTRACTS['USDT'], CONTRACTS['PLEX_ONE']]
        result1 = self._get_amounts_out_with_retry(amount_in, pool1_path)
        
        # Если прямой не сработал, пробуем через WBNB
        wbnb_path = [CONTRACTS['USDT'], CONTRACTS['WBNB'], CONTRACTS['PLEX_ONE']]
        result2 = self._get_amounts_out_with_retry(amount_in, wbnb_path)
```

### 2. Альтернативные пути обмена

Приложение автоматически пробует следующие пути:

**Для покупки PLEX ONE за BNB:**
- `BNB → WBNB → PLEX ONE` (прямой)
- `BNB → WBNB → USDT → PLEX ONE` (через USDT)

**Для покупки PLEX ONE за USDT:**
- `USDT → PLEX ONE` (прямой)
- `USDT → WBNB → PLEX ONE` (через WBNB)

## 🔧 Технические детали

### Обработка ошибок
Приложение корректно обрабатывает случаи отсутствия пулов:

```python
if not amounts_out or amounts_out[-1] == 0:
    self.log("❌ Нет ликвидности для выбранного пути", "ERROR")
    return {'success': False, 'error': 'Нет ликвидности для обмена'}
```

### Логирование
Все попытки поиска пулов логируются для отладки:

```python
self.log(f"🔍 Проверяем известные пулы PLEX ONE - USDT...", "INFO")
self.log(f"✅ Найден путь USDT->WBNB->PLEX! Результат: {result2}", "SUCCESS")
```

- Количество успешных обменов через WBNB
- Количество fallback на альтернативные пути
- Ошибки "пул не найден"
- Время выполнения операций обмена
